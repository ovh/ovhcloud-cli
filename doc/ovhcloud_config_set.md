## ovhcloud config set

Set a value in the CLI configuration

```
ovhcloud config set <configuration key> <configuration value>
```

### Examples

```
ovhcloud config set default_cloud_project xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

### Options

```
  -h, --help   help for set
```
