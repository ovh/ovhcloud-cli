## ovhcloud config set-endpoint

Configure CLI to use the given API endpoint (EU, CA, US), or a specific URL (e.g. https://eu.api.ovh.com/v1)

```
ovhcloud config set-endpoint <region>
```

### Examples

```
ovhcloud config set-endpoint EU
```
